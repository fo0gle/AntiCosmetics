# AntiCosmetics
Simple anti-cosmetics plugin for unturned rocketmod/Ldm 

https://github.com/user-attachments/assets/78d4f98d-9a94-4a71-be4c-d79abc1f4a0f

The config file is 

```xml
<?xml version="1.0" encoding="utf-8"?>
<AntiCosmeticConfiguration xmlns:xsd="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
  <RemoveHat>true</RemoveHat>
  <RemoveGlasses>true</RemoveGlasses>
  <RemoveMask>true</RemoveMask>
  <RemoveShirt>true</RemoveShirt>
  <RemovePants>true</RemovePants>
  <RemoveVest>true</RemoveVest>
  <RemoveBackpack>true</RemoveBackpack>
  <CheckIntervalSeconds>5</CheckIntervalSeconds>
  <BypassIds 76561199089826902 />
</AntiCosmeticConfiguration>
```
